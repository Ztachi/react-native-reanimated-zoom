# @repo/image-preview

## 0.1.0

### Minor Changes

- Initial release of `@repo/image-preview`.
- `ZoomableImage` component with pinch-to-zoom, momentum pan, rubber-band overscroll and double-tap focal zoom.
- Auto-detects image dimensions via `Image.getSize` when `imageWidth`/`imageHeight` are not provided.
- Fully configurable via `config` prop with sensible defaults.
